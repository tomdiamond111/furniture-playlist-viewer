# FirstTaps MV3D Furniture Viewer

Web viewer for shared furniture playlists from the FirstTaps MV3D app.

## Live Site
https://share.firsttaps.com

## Features
- Clean, modern UI
- Displays furniture playlists with all objects
- Shows statistics (YouTube videos, Spotify tracks, etc.)
- Mobile-responsive design
- Supports paste.ee and JSONbin.io URLs

## Technical Details
- Static HTML/CSS/JavaScript
- No build process required
- Hosted on GitHub Pages with custom domain
- Fetches furniture data from paste services via API
